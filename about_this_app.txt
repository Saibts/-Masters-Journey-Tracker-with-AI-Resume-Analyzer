ABOUT: MASTERS & ACADEMICS TRACKER

The Masters & Academics Tracker is a premium, client-side, local-first web application designed for students managing undergraduate coursework and planning higher education journeys abroad. Built with a focus on absolute privacy, security, and a rich vintage aesthetic, it offers a personal workspace that runs entirely in the browser.

========================================================================
1. CORE PORTALS
========================================================================

The application splits into two dedicated workspaces, tailored for distinct phases of an academic career:

[A] Undergrad Academic Portal
Designed to manage current coursework, semester-wise tracking, and undergraduate profiles:
* Academic Grades & CGPA: Track subject names, credits, and grade points semester-by-semester.
* Anna University Mapping: Translates letter grades (O, A+, A, B+, B, C) directly into the official grade point system for seamless GPA/CGPA computation.
* Academic Projects Catalog: Document undergraduate capstone, mini-projects, research projects, role descriptions, and tech stacks.
* Transcript & Report Card Generator: Generate clean, printable PDF-friendly report cards and transcripts of your academic record.

[B] Master's Journey Tracker
Designed for higher education planning, international university discovery, and admission tracking:
* College Discovery: Search and match with major international graduate programs based on your B.E. discipline, CGPA, and country preferences.
* AI Resume Intelligence: Parse and upload resumes to extract academic keywords, run compatibility matches against target university profiles, and identify key skillset gaps.
* Active Roadmaps: Follow interactive checklists covering GRE prep, English proficiency tests (IELTS/TOEFL), LOR procurement, and SOP drafting.
* Application Tracker (Kanban): Track your graduate school applications through a visual board split by status: Interested, In Progress, Submitted, and Decisions.
* Research Assistant: Integrated scientific lookup to search publications, read literature metadata, and bookmark academic sources.

========================================================================
2. SECURITY, PRIVACY & LOCAL-FIRST ARCHITECTURE
========================================================================

* 100% Private Offline Storage: All user profiles, grades, resume keywords, and application statuses are stored strictly inside the browser's localStorage. No data is uploaded to external clouds, ensuring full protection against privacy leaks.
* Tabbed Security Sign-In: Users register with a unique ID and password.
* Safe Name Registration: If a user attempts to register a User ID that is already taken locally, the system prevents overwrite and throws the warning: "already taken try another id".
* Credential Upgrades: Older profiles without passwords automatically prompt password creation upon login to update their local security.
* Danger Zone Deletion: Dedicated account deletion permanently purges all profile metrics, grades, and application roadmaps from the local browser storage.

========================================================================
3. SECURITY AUDIT & PRIVACY ASSURANCE
========================================================================

A codebase-wide security audit confirms the following protections:

* Zero Outgoing Network Traffic: No external network libraries (such as Axios) or native browser fetch calls are present in the core application logic. 
* Rule-Based Resume Intelligence: The parsing engine runs entirely in the client-side JavaScript runtime using pdfjs-dist. All keyword classification and skill-gap mappings are performed locally via predefined rule sets rather than delegating to cloud-based LLM APIs or external AI engines.
* Zero Leakage to AI/Resources: Because the app is completely self-contained and network-free, no private user metrics, profile data, or document contents are ever transmitted to any external AI services or databases.

4. THEME & VINTAGE AESTHETICS
========================================================================
